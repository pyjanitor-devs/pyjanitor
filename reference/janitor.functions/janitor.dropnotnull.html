janitor.dropnotnull¶

janitor.dropnotnull(df: pandas.core.frame.DataFrame, column_name: Hashable) → pandas.core.frame.DataFrame[source]¶

Drop rows that do not have null values in the given column.

This method does not mutate the original DataFrame.

Example usage:

df = pd.DataFrame(...).dropnotnull('column3')
Parameters

  • df – A pandas DataFrame.

  • column_name – The column name to drop rows from.

Returns

A pandas DataFrame with dropped rows.
