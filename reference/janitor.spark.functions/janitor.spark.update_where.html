janitor.spark.update_where¶

janitor.spark.update_where(df: pyspark.sql.dataframe.DataFrame, conditions: str, target_column_name: str, target_val: Union[str, int, float]) → pyspark.sql.dataframe.DataFrame[source]¶

Add multiple conditions to update a column in the dataframe.

This method does not mutate the original DataFrame.

Example usage:

import pandas as pd
from pyspark.sql import SparkSession
import janitor.spark

spark = SparkSession.builder.getOrCreate()

data = {
    "a": [1, 2, 3, 4],
    "b": [5, 6, 7, 8],
    "c": [0, 0, 0, 0]
}
df = spark.createDataFrame(pd.DataFrame(data))
df = (
    df
    .update_where(
        conditions="a > 2 AND b < 8",
        target_column_name='c',
        target_val=10
    )
)
df.show()
# +---+---+---+
# |  a|  b|  c|
# +---+---+---+
# |  1|  5|  0|
# |  2|  6|  0|
# |  3|  7| 10|
# |  4|  8|  0|
# +---+---+---+
Parameters

  • df – Spark DataFrame object.

  • conditions – Spark SQL string condition used to update a target column and target value

  • target_column_name – Column to be updated. If column does not exist in dataframe, a new column will be created; note that entries that do not get set in the new column will be null.

  • target_val – Value to be updated

Returns

An updated spark DataFrame.
